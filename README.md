# Agent2Agent (A2A) protocol

### Resources
- [GitHub repository](https://github.com/nirgluzman/a2a-tutorial.git)
- [Agent2Agent (A2A) protocol documentation](https://a2a-protocol.org/latest/)
- [Google A2A tutorial](https://codelabs.developers.google.com/intro-a2a-purchasing-concierge#0)
- [A2A YouTube course](https://www.youtube.com/playlist?list=PL6tW9BrhiPTCKTXXJAwigi7QDNpA7t4Ip)

A2A protocol defines a standardized way for AI agents to interact, enabling them to exchange messages, negotiate tasks, and share information effectively.<br/>
This protocol is designed to facilitate collaboration between agents, allowing them to work together towards common goals.
